What is GitOps

What is? A way to mange infrastructure by using Git as the single control centre. Instead of manually setting up or updating the system, config file that describe the system is stored on Git. When these files are change, automation tools (Argo CD or Flux) will automatically update the systems to match the new config. It makes the deployment consistent, easy to track (version control), and quick to roll back if something goes wrong. ...
